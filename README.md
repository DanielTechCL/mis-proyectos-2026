# cohetes con funciones
Este es un programa de python, en el cual le pide al usuario la cantidad de cohetes que quiere lanzar la velocidad y el tiempo de viaje
Luego de hacer las preguntas, el programa envia al usuario a una pagina web en la que se muestra el grafico con las instrucciones que el usuario dio al programa
## tecnologias
python
plotly graph
numpy
